## How to install

```
cd VulnPlayground 
cd wizard-sorcerer
sudo docker build -t wizard-sorcerer .
sudo docker run -d -p 80:80 --name wizard-sorcerer wizard-sorcerer
```

You will reach the website by going to `http://127.0.0.1`
